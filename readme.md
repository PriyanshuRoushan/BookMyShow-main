## 🌟 Project Overview

This project replicates the core functionality of **BookMyShow-main**, allowing users to:

- 🎬 Browse movies  
- 🕒 View showtimes  
- 🎟️ Book seats in real-time  
- ⚡ Real-time updates using Socket.IO  

Built using **MERN Stack** with scalable architecture.

## 📡 API Endpoints

### 🔐 Authentication
| Method | Endpoint | Description |
|-------|----------|------------|
| POST | /api/auth/register | User onboarding |
| POST | /api/auth/login | JWT authentication |

### 🎟️ Movies & Booking
| Method | Endpoint | Description |
|-------|----------|------------|
| GET | /api/movies | Fetch all movies |
| GET | /api/movies/:id | Movie details |
| POST | /api/bookings | Book seats |
